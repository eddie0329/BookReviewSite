# 너도 읽을래?

사용자들끼리 책의 후기를 남기는 사이트입니다.

## Stack

- React
- Redux
- Redux Thunk
- ant design
- jest
- enzyme
- react router

## Start

```script
git clone https://github.com/eddie0329/BookReviewSite.git
cd BookReviewSite
npm i
```

## Development

```script
npm start
```

## Deploy

```script
npm run build
```
